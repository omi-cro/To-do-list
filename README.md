# To-Do List Application (Console Based)

## Description
This is a simple console-based To-Do List application developed using the C programming language.  
It allows users to add tasks and view the list of added tasks.

## Features
- Add new tasks
- View all tasks
- Simple menu-driven interface

## Technologies Used
- C Programming Language

## How to Run
1. Compile the program:
   gcc todo.c -o todo
2. Run the executable:
   ./todo

## Purpose
This project is created to practice basic input handling, array usage, and simple program flow.
